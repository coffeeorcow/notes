## 为什么需要学习 vim
    
1. vim 可以在终端中使用
2. vim 可以对文本进行快速的浏览、修改等操作，并且还可以配合各种插件搭建各种不同的应用场景（如整合git，添加目录管理，增加 Java 提示等）
3. 许多编辑器都对 vim 有很不错的支持，如 Intellij IDEA, Visual Studio Code, Atom 等等，减少记忆很多不同的快捷键

## Vim 核心

vim 主要有以下几种模式:
1. Normal Mode 普通模式 (ESC 或 ctrl+[ 进入)
2. Visiual Mode 可视模式 (普通模式下按 v 进入)
3. Insert Mode 插入模式 (普通模式下按 i, a, o 进入)
4. Command-Line Mode 命令模式 (普通模式下按 : 进入)
5. Ex Mode ex 模式 (普通模式下按 Q 进入)
